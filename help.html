Database Help:

Deleting a Row

To delete a row, follow these instructions:

  1. Under the "Manage" tab in the top navigation bar, select the "Delete Item" option.

  2. Select the table you wish to delete the key from.

  3. Type in the attribute of the row you wish to delete.

  4. Under Where:, type in the attribute you wish to compare to with your conditional.

  5. Type in an Operation: >, <, =, !=, etc.

  6. Under Conditional, type in what you're comparing to. Ex) number > 3.

  7. Finally, select CASCADE or SET NULL for what type of deletion you want to do.


Modify an Attribute

To modify an attribute, follow these instructions:

  1. Under the "Manage" tab in the top navigation bar, select the "Modify Item" option.

  2. Select the table you wish to modify the key from.

  3. Enter the attribute you wish to change.

  4. Enter the value you wish to change the attribute to. Put quotation marks around Strings.

  5. Enter the attribute name. No quotation marks.

  6. Enter the operation. >, <, =, !=, etc.

  7. Enter the value you wish to compare to. Use quotation marks if it's a string.


View a Row

To view a Row, follow these instructions:

  1. Under the "Manage" tab in the top navigation bar, select the "View Item" option.

  2. Select the table you wish to view the key from.

  3. Enter the attribute name. No quotation marks.

  4. Enter the operation. >, <, =, !=, etc.

  5. Enter the value you wish to compare to. Use quotation marks if it's a string.
